# 🎯 Speed Sniper
> Upload a resource. Snipe the correct answer.

## What is Speed Sniper?
Speed Sniper is a fast-paced AI-powered quiz game.
8 answer tokens float across the screen.
Tap the correct one before it drifts off!

## Features
- 📄 Upload PDF or paste text
- 🤖 AI generates 8 rounds of questions
- ⚡ Tokens get faster every round
- 🔥 Combo multipliers (x1.5, x2.0)
- 🎯 3-6 minute sessions

## Tech Stack
- React + TypeScript
- Vite + NX Monorepo
- OpenRouter API
- Deployed on Vercel

## Getting Started
1. Clone the repo
2. Run `npm install`
3. Add `VITE_OPENROUTER_API_KEY` to `.env`
4. Run `npx nx serve speed-sniper-host`
